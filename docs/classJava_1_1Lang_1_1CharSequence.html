virtual char Java::Lang::CharSequence::charAt ( int  index) const
pure virtual

Returns the char value at the specified index. An index ranges from zero to length() - 1. The first char value of the sequence is at index zero, the next at index one, and so on, as for array indexing.

If the char value specified by the index is a "{@docRoot}/java/lang/Character.html#unicode">surrogate, the surrogate value is returned.
